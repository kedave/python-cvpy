Overview
--------

This high-level Python library assists with the image and biomedimage action sets by providing helpful visualization APIs. 
These APIs allow users to easily be able to move images out of CAS and into common open source tools such as Mayavi and Matplotlib. 
This streamlines the visualization of image data fetched from a CAS table and allows for further image analysis.
